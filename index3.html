Data Type

1.Every variable in Java program should have data type.

2.Java is strongly typed programming language(type checking is very important).

3.In Java every variable and every expression has some type.

4.Every assignment should be checked by compiler for typed compatability.




** Except char and boolean data type the remaining data type are considered as sign data type.


byte

size: 1 byte(8 bits)

MAX_value:+127

MIN_value:-128

Range:-128 to 127

short

size: 2 byte(16 bits)

MAX_value:+32767

MIN_value:-32768

Range:-32768 to 32767

int

size: 4 byte(32 bits)

MAX_value:+2147483647

MIN_value:-2147483648

Range:-2147483648 to 2147483647

long

size: 8 byte(64 bits)

MAX_value:2^63-1

MIN_value:-2^63

Range:-2^63 to 2^63-1

float

size: 4 byte(32 bits)

MAX_value:+3.4e38

MIN_value:-3.4e38

Range:-3.4e38 to 3.4e38

double

size:8 byte(64 bits)

MAX_value:+1.7e38

MIN_value:-1.7e38

Range:-1.7e38 to 1.7e38

boolean

size: not applicable

Range: not applicable(only allowed value are true or false)

char

size: 2 byte(16 bits)

MAX_value:65535

MIN_value:0

Range:0 to 65535
